# Validator Genesis Transactions (Gentx) Registry

Mainnet validator operators should submit their genesis staking transaction file (`gentx-*.json`) to this directory via pull request.

## How to generate your gentx:
Follow instructions in [Validator Onboarding Playbook](file:///Users/majedurrahman/Sovereign/doc/mainnet/validator-onboarding.md) step 4.3.

## Naming convention:
`gentx-<moniker>.json` (e.g., `gentx-sovereign-node-0.json`)
