# Sovereign L1 Custom Unified Explorer

This directory houses the Next.js 14 frontend for the Sovereign L1 Custom Enterprise Blockchain Explorer.

## Setup & Running Locally

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Start the development server:
   ```bash
   pnpm dev
   ```

Refer to [sovereign-l1-explorer-plan.md](../sovereign-l1-explorer-plan.md) for the full architectural specifications.
